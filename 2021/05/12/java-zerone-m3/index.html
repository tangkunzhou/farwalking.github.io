初识Spring Boot，搭建一个简单的Spring Boot项目

1.Spring Boot简介

  Spring Boot基于Spring 4.0设计，是由Pivotal公司开发的框架。2014年Spirng Boot 1.0发布（基于Spring 4.0），2018年Spring Boot 2.0发布（基于Spring 5.0）。

  • Spring Boot基于Spring开发，其设计目的是用来简化Spring应用的初始搭建以及开发过程；
  • 关于Spring Boot有一句很出名的话“预定大于配置”；
  • Spring Boot盛行的最重要的原因是微服务。
